## <a name="enum-base-type"></a>列舉基底類型

在C# 6.0 中，我們放寬了列舉基底類型的語法，使其成為類型語法，但必須系結至其中一組指定的類型。 因此允許 `System.Int32`。 但規格尚未更新;它仍然需要基底的一組關鍵字。

這是該變更規格的預留位置。
