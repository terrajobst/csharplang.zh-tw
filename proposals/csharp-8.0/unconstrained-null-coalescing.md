## <a name="unconstrained-type-parameter-in-null-coalescing-operator"></a>Null 聯合運算子中的不受限制型別參數

在C# 8.0 中，我們導入了一項功能，可讓 null 聯合運算子具有左運算元，但這不是參考或實值型別（亦即不受限制的型別參數）。 這是其規格的預留位置。
