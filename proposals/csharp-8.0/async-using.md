## <a name="async-using-declaration"></a>非同步 using 宣告

在C# 8.0 中，我們新增了*非同步 using*語句的支援。 有兩種形式。 其中一個具有使用 declaratation 之範圍的區塊主體。 另一個宣告為本機，而且會以隱含的範圍限定在區塊的結尾。 這是其規格的預留位置。
